# Server Performance Monitoring Script

This project is a Bash script that analyzes basic Linux server performance statistics.

## Features

- CPU Usage
- Memory Usage
- Disk Usage
- Top 5 CPU consuming processes
- Top 5 memory consuming processes
- OS Version
- System Uptime
- Load Average
- Logged in Users
